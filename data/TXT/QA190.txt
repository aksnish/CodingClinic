Question: How should an aspiration via a lumbar drain port be reported? What body part character should we use for the lumbar drain port? 
Answer: The body part value for the lumbar port should be “spinal canal.” Aspiration via a lumbar drain port could be either therapeutic or diagnostic. The ICD10-PCS code assignment would be dependent on the objective of the procedure (therapeutic or diagnostic). If the procedure is diagnostic use the qualifier “X” and if therapeutic use qualifier “Z.” The “approach” is percutaneous, when the aspiration is done via a port. Currently, the ICD-10PCS Official Guidelines for Coding and Reporting indicate the qualifier “X” may only be used for procedures documented as biopsies, but the next version of the guidelines will be revised, so that qualifier “X” may be used whenever the objective of the procedure is diagnostic.
